    1. **`src/Hooks/useDebounce.js`**:
       - Export `useDebounce(value, delay = 300)` hook
       - Uses `useState` + `useEffect` with `setTimeout`/`clearTimeout`
       - Returns the debounced value
       - Include JSDoc comment explaining the hook

    2. **`src/Hooks/usePagination.js`**:
       - Export `usePagination({ initialPage = 1, initialLimit = 12 } = {})` hook
       - Returns: `{ page, limit, totalPages, totalItems, setPage, setLimit, setTotal, nextPage, prevPage, hasNext, hasPrev, reset }`
       - `setTotal(count, limit)` computes `totalPages = Math.ceil(count / limit)`
       - `hasNext = page < totalPages`, `hasPrev = page > 1`
       - `nextPage`/`prevPage` clamp within valid range
       - Include JSDoc comment

    3. **`src/Hooks/useSmartLoader.js`**:
       - Export `useSmartLoader({ delay = 300, minDisplay = 500 } = {})` hook
       - Manages three states: `idle`, `loading` (after delay), `minDisplay` (enforcing minimum time)
       - Accepts a `isLoading` boolean (the raw loading state from the store)
       - Returns `{ showLoader, isLoading: isActuallyLoading }`
       - Logic: When `isLoading` becomes true → start a 300ms timer; if still loading after 300ms → show loader (state = 'loading'). When `isLoading` becomes false → if currently showing, enforce minimum 500ms display before hiding.
       - This prevents flash-of-loader for fast responses (<300ms) and janky hide-show-hide for quick transitions
       - Include JSDoc comment
  </action>

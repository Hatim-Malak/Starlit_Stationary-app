**Color Palette Mapping Guide:**
Replace old blue colors with new palette:

| Old Class | New Class | Hex |
|-----------|-----------|-----|
| `bg-blue-600` | `bg-primary` or `bg-secondary` | #213448 / #547792 |
| `bg-blue-700` | `bg-primary` | #213448 |
| `from-blue-600 to-blue-700` | `from-secondary to-primary` | #547792 → #213448 |
| `from-blue-700 via-blue-600 to-blue-500` | `from-primary via-secondary to-primary` | #213448 → #547792 → #213448 |
| `text-blue-100` | `text-accent` | #94B4C1 |
| `text-blue-600` | `text-primary` or `text-secondary` | #213448 / #547792 |
| `text-blue-700` | `text-primary` | #213448 |
| `border-blue-200` | `border-accent` or `border-accent/50` | #94B4C1 |
| `border-blue-300` | `border-accent` | #94B4C1 |
| `bg-blue-50` | `bg-warm` or `bg-accent/10` | #EAE0CF / #94B4C1 |
| `bg-blue-100` | `bg-warm-100` or `bg-accent/20` | #f2ebe0 / #94B4C1 |
| `ring-blue-500`/`focus:border-blue-500` | `ring-secondary`/`focus:border-accent` | #547792 / #94B4C1 |
| `from-blue-700 to-blue-600` | `from-primary to-secondary` | #213448 → #547792 |
| `from-amber-50 to-yellow-50` | `from-warm-50 to-warm` | Keep warm-toned for info banners |
| `from-yellow-50 to-orange-50` | `from-warm-50 to-warm` | Keep warm-toned for info banners |
| `from-red-500 to-red-600` | Keep red (destructive actions) | Keep red |
| `shadow-blue-200` | `shadow-accent/30` | #94B4C1 |

**General Approach:**
- Blue → primary/secondary
- Light blue bg → warm or accent/10
- Blue text → primary (dark bg) or secondary (light bg)
- Blue borders → accent
- Yellow/gold accents → warm
- Red alerts → keep red (destructive)
- Green success → keep green (status indicators)
